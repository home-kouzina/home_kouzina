# Amazon FBA Lot Support

This module extends Odoo 18's native `sale_amazon` connector for FBA order imports.

Native Odoo creates and validates FBA stock moves directly from the Amazon account stock location to the customer location. For products tracked by lots, native logic sets the done quantity but does not assign a lot, causing:

> You need to supply a Lot/Serial Number

This module keeps the native behavior for non-tracked products and adds lot assignment for products with `Tracking = By Lots`.

## Behavior

- Applies only to Amazon FBA stock moves generated by `amazon.account._generate_stock_moves()`.
- Non-tracked products keep the native Odoo behavior.
- Lot-tracked products:
  - search existing lots from the Amazon account stock location, including child locations;
  - consume available lots in FIFO order using `stock.quant.in_date`;
  - if order quantity is greater than available quantity, the remaining quantity is assigned to the last selected lot, allowing negative stock if the product/company configuration allows it;
  - if no lot exists in the Amazon location for the product, a clear error is raised.
- Serial-number tracking is intentionally not handled.

## Install

1. Copy `sale_amazon_lot` into your Odoo custom addons path.
2. Restart Odoo.
3. Update Apps List.
4. Install **Amazon FBA Lot Support**.
5. Retry Amazon order recovery/sync.

## Important

This module does not fix Amazon Offer/SKU mapping. Amazon Offers should still be mapped correctly before bulk order sync.
